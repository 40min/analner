
##Installation

* from src
    * clone this repo
    * run
``` pip install -r ./requirements.txt ```
    * setup environment variables
        * DATA_PATH -- dir where will be stored text headers and models in json

* as pip pkg from git:
    * pip install -e git+https://github.com/40min/analner.git@master#egg=analner
